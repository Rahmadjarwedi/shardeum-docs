# Running a archive

## Why run an archive?

Running your own archive:

    -helps track Shardeum's transaction history
    -earns SHM for storing Shardeum's transaction history
    -send 1 ethereum to this address : 0xCD5D898A1CC07aA0f5EE2d1210802bc329255108
